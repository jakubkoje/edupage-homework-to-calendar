# edupage-homework-to-calendar

---
## Add EduPage homeworks to your google calendar
### Usage:
1. clone the repo
2. npm i
3. create .env and add these fields
   - EDUPAGE_NAME=
   - EDUPAGE_PASSWORD=
   - EDUPAGE_CALENDAR_ID=
4. generate credentials.json and add it to public folder
5. tsc
6. node dist/index.js
